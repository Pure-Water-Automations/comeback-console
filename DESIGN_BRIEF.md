# Operation COMEBACK — Design Brief for New Routes

This app is a scroll-driven narrative engine (TanStack Start + React 19 + Tailwind v4 +
`motion/react`). New routes MUST match the existing aesthetic. Study
`src/components/game/CosmicRules.tsx` and `src/styles.css` before writing code.

## Visual language (non-negotiable)

- **Dark cosmic field**: page background `#0a0a0b`, parallax starfield layers with the
  `animate-twinkle` class, occasional shooting star, per-section radial glow backdrops
  (`radial-gradient(circle at 50% 20%, rgba(79,127,255,0.2), transparent 35%)` — rotate
  hues per section: blue, teal, gold, violet...).
- **Swiss type on a space field**: Helvetica. Big display headlines (`.display` class /
  `font-bold tracking-[-0.04em]`, sizes like `text-[10vw]`), uppercase kickers with
  `text-sm uppercase tracking-[0.4em]`, numbered indexes ("01 ·", "02 ·").
- **Zero border radius** (`--radius: 0rem`). Cards = `border border-white/10` +
  `bg-black/60 backdrop-blur-md`. Glows via `shadow-[0_0_12px_...]`.
- **Pixel sprites**: assets in `src/assets/sprites/<family>/<pose>.png` (families:
  adventurer, mentor, npc, smart_guy, spirit, wizard — 6 poses each). Render with
  `[image-rendering:pixelated]` + `drop-shadow-2xl`, bob with
  `animate={{ y: [0, -10, 0] }} transition={{ duration: 0.9, repeat: Infinity }}` or the
  `animate-float` class. Import PNGs as ES modules.
- **Motion idioms**: `useScroll` + `useTransform` + `useSpring({ stiffness: 80, damping: 24, mass: 0.4 })`,
  reveals with `initial={{ opacity: 0, y: 20 }}`, ease `[0.16, 1, 0.3, 1]`.
- Accent token `text-signal` exists; chart colors via `--chart-1..5`.

## Data layer (already written — use it, don't duplicate)

`src/lib/comebackData.ts` exports:
- `COMMUNITIES`, `TRIMESTER`, `TEAM_LABELS`
- `rankedCommunities()` → sorted standings with `rank`, `points`, per-category points
- `pctOfTarget`, `categoryPoints`, `totalPoints`
- `communityBadges(c)`, `communityLevel(points)`, `coachingTips(c)`

Scoring rule (real campaign math): points = (percent-of-target − 100) × 10.

## Rules of engagement

- Routes live in `src/routes/<name>.tsx` using `createFileRoute` (copy the pattern from
  `src/routes/index.tsx`, including the `head()` meta).
- **Do NOT edit `src/routeTree.gen.ts`** — it is generated by the router plugin.
- Components go in `src/components/game/<feature>/`.
- Use existing shadcn/ui components from `src/components/ui/` where sensible
  (Progress, Tabs, Select, Card) but restyle to match the dark cosmic look.
- `recharts` is available for charts — style with dark grid lines (`stroke="#ffffff14"`)
  and chart token colors.
- TypeScript strict; `cn()` util from `@/lib/utils`; path alias `@/` = `src/`.
- Verify with `npm run build` before finishing.
